VALO_Rank_Overlay(Ver 1.1.0)  readme.txt

※使用前に必ずお読みください。

	ダウンロードいただき、ありがとうございます。
	VALO_Rank_Overlayは、配信などで使用する、VALORANTのランクを表示するものです。

________________________________________________________________________________
1.インストール / アンインストール方法

	インストールなどは特に必要ありません。不要になれば削除してください。

________________________________________________________________________________
2.使用方法

	2.a.script.jsを編集
		script.jsをテキスト形式で開きます。
		Regiao,Username,Taglineの内容を自分に合わせて変更します。
		例：変更前：let Username = "君にリコン届け";	//名前
		　：変更後：let Username = "君に離婚届";	//名前

		index.htmlを使用して、自分のランクが表示されれば1.aはOKです。
		index.htmlをブラウザで表示させてもいいですし、OBSなどの配信用ソフトウェア
		に適用して確認しても問題ありません。

	2.b.index.htmlをOBSなどの配信用ソフトウェアなどに適用(OBSを例に)
		index.htmlをOBSのソース欄にドラッグ＆ドロップすれば適用できます。
		幅は1400、高さは400以上に設定してください。表示が変になる場合があります。
